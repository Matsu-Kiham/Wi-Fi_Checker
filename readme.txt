Wi-Fiチェッカー

製作者：Matsu-Kiham
バージョン：1.00

指定した範囲のIPアドレスと外部機器との接続状況を確認し，
未登録のMACアドレスを持つ外部機器が検出された場合，
それを報告しログを記録するWindowsアプリです．

導入方法：
まず，空のフォルダを用意します．フォルダの場所はどこでも
構いません．次に，以下のファイルをダウンロードして，
用意したフォルダに入れてください．
・Auto.txt
・ip.txt
・MAC_list.txt
・MAC_log.txt
・MAC_log_backup.txt
・Wi-Fi Checker.exe

そのあと，Wi-Fi Checker.exeを実行すると，アプリが起動します．
なお，以下のファイルはプログラムのソースコードとアイコンです．
・Form1.cs
・Form1.Designer.cs
・Form2.cs
・Form2.Designer.cs
・Form3.cs
・Form3.Designer.cs
・Form4.cs
・Form4.Designer.cs
・favicon.ico


アプリの操作法：
１．IPアドレスの登録
”IPアドレス登録”と書かれたボタンをクリックします．すると，
IPアドレスの登録画面が表示されるので接続状況を確認したい
IPアドレスを入力して，登録ボタンをクリックしてください．
例：接続状況を確認したいIPアドレスの範囲が192.168.11.2から
192.168.11.65までであれば，上側の入力部分に192.168.11.2を
入力し，下側の入力部分に192.168.11.65を入力する．

２．MACアドレスの登録
”MACアドレス登録”と書かれたボタンをクリックします．すると，
MACアドレスの登録画面が表示されるので，接続スキャン時に
ログに記録しないMACアドレスを入力して登録ボタンをクリック
してください．MACアドレスは半角入力で，アルファベットは
小文字で入力してください．
例：32-52-98-2g-y2-a6

３．接続スキャン
”接続スキャン”と書かれたボタンをクリックすると，登録した
IPアドレスの接続状況の確認が始まります．確認が始まると，
”接続スキャンを実行中です．このアプリを操作しないでください．”
というメッセージが”接続スキャン”と書かれたボタンの下に
表示されます．接続状況の確認が終わるとこのメッセージは
消えます．未登録のMACアドレスが検出されていた場合，
効果音が鳴り，”未登録のMACアドレスの接続がありました”と
表示されます．

４．ログの表示
”ログ表示”と書かれたボタンをクリックすると，アプリ下部の空欄
に未登録のMACアドレスの検出記録が表示されます．

５．自動スキャンのOn/Off
”自動スキャンOff”と書かれたボタンをクリックすると，
”自動スキャンOn”という表示になり，以後30分おきに接続スキャン
が実行されます．”自動スキャンOn”と書かれたボタンをクリックすると，
”自動スキャンOff”という表示になり，以後接続スキャンは自動で実行
されなくなります．

６．MACアドレスの登録解除
”MACアドレスの登録解除”と書かれたボタンをクリックします．すると，
登録済みのMACアドレスが表示されるので，登録を解除したい
MACアドレスをクリックして登録解除ボタンをクリックしてください．
